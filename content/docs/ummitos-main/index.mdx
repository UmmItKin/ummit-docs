### UmmItOS Overview

UmmItOS is a GNU/Linux distribution based on Arch Linux and Hyprland DWM, featuring a personal configuration setup and dotfiles collection.

### Current Status

- **Development Stage**: Heavy development
- **Available**: Configuration files and installation script
- **Planned**: ISO release in the coming weeks
- **Bugs**: Bugs are expected as I'm still working on it. If you encounter any issues, please report them to the [GitHub repository](https://github.com/UmmItKin/Dotfiles/issues)

### Repository

You can view the source code at: [GitHub repository](https://github.com/UmmItKin/Dotfiles)
