## Quick reference

### Main modifier key

- **$mainMod = SUPER** (Windows key)

## Configuration files

All keybindings are configured in the following files:
- **Launcher Config**: `./configs/hypr/hyprland/launcher.conf`
- **Keybinds Config**: `./configs/hypr/hyprland/hypr_keybinds.conf`
- **Laptop Config**: `./configs/hypr/hyprland/laptop.conf`
- **Location**: `~/.config/hypr/hyprland/` (after installation)

## Applying the changes

Hyprland come with auto reload feature. so you don't need to reload the configuration manually, the result will be shown immediately.
