## Configuration files location

All UmmItOS configuration files are located in:
- **Main config**: `~/.config/` (for all the configurations)
- **Script**: `~/script/` (for all the scripts)
- **Wallpaper**: `~/.wallpaper/` (for all the wallpapers)
