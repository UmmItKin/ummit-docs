## Keybindings

UmmItOS comes with many keybindings by default. Here is the list of keybindings. If you want to modify the keybindings, you can modify the `./configs/hypr/hyprland/launcher.conf` file. For laptop configuration, you can refer to the `./configs/hypr/hyprland/laptop.conf` file.

The `launcher.conf` file contains 3rd party application keybindings, which are very useful for your daily workflow. The `hypr_keybinds.conf` file contains keybindings built for Hyprland - no additional software is required for these keybindings.

I've modularized the files to make it easy to identify the keybindings and their actions, so you can easily modify the keybindings to your liking.


## Hyprland Built-in Features

```shell
# This file only contains the keybindings for the Hyprland built-in features

$mainMod = SUPER

bind = $mainMod, C, killactive, 
bind = $mainMod, M, exit, 
bind = $mainMod, V, togglefloating, 
bind = $mainMod, J, swapnext,

# Move focus with mainMod + arrow keys
bind = $mainMod, left, movefocus, l
bind = $mainMod, right, movefocus, r
bind = $mainMod, up, movefocus, u
bind = $mainMod, down, movefocus, d

# Switch workspaces with mainMod + [0-9]
bind = $mainMod, 1, workspace, 1
bind = $mainMod, 2, workspace, 2
bind = $mainMod, 3, workspace, 3
bind = $mainMod, 4, workspace, 4
bind = $mainMod, 5, workspace, 5
bind = $mainMod, 6, workspace, 6
bind = $mainMod, 7, workspace, 7
bind = $mainMod, 8, workspace, 8
bind = $mainMod, 9, workspace, 9
bind = $mainMod, 0, workspace, 10

# Move active window to a workspace with mainMod + SHIFT + [0-9]
bind = $mainMod SHIFT, 1, movetoworkspace, 1
bind = $mainMod SHIFT, 2, movetoworkspace, 2
bind = $mainMod SHIFT, 3, movetoworkspace, 3
bind = $mainMod SHIFT, 4, movetoworkspace, 4
bind = $mainMod SHIFT, 5, movetoworkspace, 5
bind = $mainMod SHIFT, 6, movetoworkspace, 6
bind = $mainMod SHIFT, 7, movetoworkspace, 7
bind = $mainMod SHIFT, 8, movetoworkspace, 8
bind = $mainMod SHIFT, 9, movetoworkspace, 9
bind = $mainMod SHIFT, 0, movetoworkspace, 10

# Magic Hidden workspaces
bind = $mainMod, S, togglespecialworkspace, magic
bind = $mainMod SHIFT, S, movetoworkspace, special:magic

# Scroll through existing workspaces with mainMod + scroll
bind = $mainMod, mouse_down, workspace, e+1
bind = $mainMod, mouse_up, workspace, e-1

# Move/resize windows with mainMod + LMB/RMB and dragging
bindm = $mainMod, mouse:272, movewindow
bindm = $mainMod, mouse:273, resizewindow

# sets repeatable binds for resizing the active window
binde = ALT_L, right, resizeactive, 5 0
binde = ALT_L, left, resizeactive, -5 0
binde = ALT_L, up, resizeactive, 0 -5
binde = ALT_L, down, resizeactive, 0 5
```

## Laptop Keybindings

```shell
# These keybindings usually work on laptops.

# Brightness control with brightnessctl
bind = , XF86MonBrightnessUp, exec, brightnessctl set +1%
bind = , XF86MonBrightnessDown, exec, brightnessctl set 1%-

# Volume control with wpctl, package from wireplumber
bindel = , XF86AudioRaiseVolume, exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 1%+
bindel = , XF86AudioLowerVolume, exec, wpctl set-volume @DEFAULT_AUDIO_SINK@ 1%-
bindl = , XF86AudioMute, exec, wpctl set-mute @DEFAULT_AUDIO_SINK@ toggle

# playerctl for media control
bindl = , XF86AudioPlay, exec, playerctl play-pause
bindl = , XF86AudioPrev, exec, playerctl previous
bindl = , XF86AudioNext, exec, playerctl next

# Launch the ollama container with ALT_R + Copilot key
bind = ALT_R, code:133, exec, kitty -e ollama run llama3.1:latest
```

## Third-Party Applications

```shell
# This file only contains the keybindings for 3rd party applications

$key = TAB
$modifier = ALT
$modifier_release = ALT_L
$reverse = SHIFT

# Define the applications to be used
$terminal = kitty
$fileManager = kitty -e yazi /home/$USER

# Define the scripts to be used
$application_launcher = bash ~/script/rofi/application_launcher.sh
$clipboard_manager = bash ~/script/cliphist/cliphist-menu.sh
$wallpaper_picker = bash ~/script/swww/swww.sh
$color_picker = bash ~/script/hypr/hyprpicker/hyprpicker.sh
$lock_screen = hyprlock

# binding the programs to the keys
bind = $mainMod, T, exec, $terminal
bind = $mainMod, E, exec, $fileManager
bind = $mainMod, 36, exec, $application_launcher
bind = $mainMod, X, exec, bash ~/script/wlogout/blur-background.sh
bind = $mainMod, L, exec, $lock_screen
bind = ALT_L, V, exec, $clipboard_manager
bind = ALT_L, W, exec, $wallpaper_picker
bind = ALT_L, E, exec, smile
bind = ALT_L, P, exec, $color_picker
bind = ALT_L, O, exec, woomer
bind = $mainMod Shift, 36 , exec, kitty -e  ~/script/waybar/update.sh

# HyprShot <- https://github.com/Gustash/Hyprshot
bind = $mainMod, PRINT, exec, hyprshot -m window
bind = , PRINT, exec, hyprshot -m output
bind = $shiftMod, PRINT, exec, hyprshot -m region -z

bind = $mainMod, F1, exec, kando --menu "Example Menu"

# Hyprswitch configuration

$key = super_l
$mod = super
bind = $mod, $key, exec, $bin gui --mod-key $mod --key $key --max-switch-offset 9

# Keyboard workspace Monitor
bind = alt, tab, exec, hyprswitch gui --mod-key alt_l --key tab --close mod-key-release --reverse-key=key=grave --switch-type=workspace && hyprswitch dispatch
bind = alt, grave, exec, hyprswitch gui --mod-key alt_l --key tab --close mod-key-release --reverse-key=key=grave --switch-type=workspace && hyprswitch dispatch -r
```
